# Data - iFood Marketing Analytics

## About This Folder

This folder contains the dataset used in the iFood Marketing Analytics project.

The dataset is used as the source for SQL modeling, Python exploratory analysis and future Power BI dashboards.

---

## Expected Dataset

The expected dataset file name is:

```text
ifood_marketing.csv
```

Expected location:

```text
projetos/ifood-marketing-analytics/data/ifood_marketing.csv
```

---

## Data Usage

This dataset will be used to analyze:

- Customer profile
- Income distribution
- Purchase behavior
- Product category consumption
- Marketing campaign response
- Customer segmentation
- Business KPIs

---

## Data Privacy

The dataset must be anonymized and should not contain sensitive personal information.

This project is intended for portfolio, learning and analytical demonstration purposes.

---

## Connection With Other Project Stages

### SQL

The data is used to create tables, analytical views, KPIs and customer segmentation queries.

### Python

The data is used for exploratory analysis, feature engineering, KPI generation and chart creation.

### Power BI

The treated data will be used to build an executive dashboard with customer, consumption and campaign insights.
