# Data / Dados

This folder contains the dataset used in the iFood Marketing Analytics project.

Esta pasta contém a base de dados utilizada no projeto iFood Marketing Analytics.

## Expected file / Arquivo esperado

The Python script expects a CSV file with the following name:

O script Python espera um arquivo CSV com o seguinte nome:

```text
ifood_marketing.csv
```

## Notes / Observações

The dataset must be anonymized and should not contain sensitive personal information.

A base deve estar anonimizada e não deve conter informações pessoais sensíveis.
