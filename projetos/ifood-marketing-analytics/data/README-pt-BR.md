# Dados - iFood Marketing Analytics



## Sobre Esta Pasta

Esta pasta contém a base de dados utilizada no projeto iFood Marketing Analytics.

A base será utilizada como fonte para as etapas de SQL, análise exploratória em Python e construção futura do dashboard em Power BI.

---

## Arquivo Esperado

O arquivo esperado deve ter o seguinte nome:

```text
ifood_marketing.csv
```

Local esperado dentro do projeto:

```text
projetos/ifood-marketing-analytics/data/ifood_marketing.csv
```

---

## Uso dos Dados

Esta base será utilizada para analisar:

- Perfil dos clientes
- Distribuição de renda
- Comportamento de compra
- Consumo por categoria de produto
- Resposta a campanhas de marketing
- Segmentação de clientes
- Indicadores de negócio

---

## Privacidade dos Dados

A base deve estar anonimizada e não deve conter informações pessoais sensíveis.

Este projeto tem finalidade de portfólio, aprendizado e demonstração analítica.

---

## Conexão com as Outras Etapas do Projeto

### SQL

Os dados serão utilizados para criação de tabelas, visões analíticas, indicadores e consultas de segmentação de clientes.

### Python

Os dados serão utilizados para análise exploratória, criação de variáveis analíticas, geração de KPIs e criação de gráficos.

### Power BI

Os dados tratados serão utilizados para construção de um dashboard executivo com análises sobre clientes, consumo e campanhas.
