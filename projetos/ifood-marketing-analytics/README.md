# iFood Marketing Analytics

Portfolio project focused on customer behavior, marketing campaign performance, predictive modeling, customer segmentation and executive dashboards.

## Project Overview

This project simulates a business scenario where a food delivery and marketplace company needs to understand customer behavior, improve campaign targeting, reduce discount waste and support data-driven marketing decisions.

## Main Deliverables

- SQL data modeling and analytical queries
- Python exploratory data analysis
- Campaign response prediction model
- Advanced customer segmentation
- Executive business insights
- Power BI dashboard
