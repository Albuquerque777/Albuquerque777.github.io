# Python Analysis Outputs

This folder contains the main outputs generated by the Python analytics layer of the iFood Marketing Analytics project.

## Output Categories

### Exploratory Analysis

- Treated dataset
- Business KPIs
- Income distribution chart
- Total spending distribution chart
- Average spending by age group
- Campaign response rate by income group

### Predictive Modeling

- Model comparison metrics
- Test performance metrics
- Campaign response predictions
- Customer targeting decile analysis
- Feature importance analysis
- ROC curve
- Lift by decile
- Campaign response probability distribution
- Top customers with higher campaign response probability

### Customer Segmentation

- Customer segment classification
- Customer segment profile
- Segment action plan
- Segment size analysis
- Average spending by customer segment
- PCA customer segment visualization

### Executive Business Insights

- Business pain action plan
- Executive business insights report
- Core business metrics
- Recommended actions by business problem
- Success KPIs for marketing and CRM decisions

| File | Description |

## Business Purpose

These outputs support marketing and CRM decision-making by helping identify:

- Customers with higher probability of campaign response
- Customer segments with different business behaviors
- Retention and reactivation opportunities
- Campaign targeting priorities
- Discount optimization opportunities
- Executive insights for data-driven decision-making

## Data Science Note

The predictive model is a campaign propensity model, not a causal uplift model. To measure true incremental impact, the next step would be A/B testing or uplift modeling with treatment and control groups.
