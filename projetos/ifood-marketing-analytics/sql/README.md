# SQL Layer

This folder contains the SQL scripts used to create, clean, structure and analyze the iFood Marketing Analytics dataset.

## Execution Order

1. `01-criacao-tabelas.sql`  
   Creates the main table structure for the customer marketing dataset.

2. `02-tratamento-dados.sql`  
   Applies data cleaning, standardization and transformation rules.

3. `03-kpis-negocio.sql`  
   Calculates business KPIs for customer behavior and campaign performance.

4. `04-segmentacao-clientes.sql`  
   Creates analytical segmentation queries based on customer behavior, income, spending and campaign response.

## Main SQL Objectives

- Structure the analytical base
- Prepare data for BI analysis
- Calculate business indicators
- Support customer segmentation
- Create reusable queries for dashboard development

## Business Outputs

Examples of business outputs generated by the SQL layer:

- Total customers
- Average income
- Average spending
- Campaign response rate
- Customer distribution by income group
- Customer distribution by age group
- Spending by product category
- Purchase channel analysis

## Tools

The scripts can be adapted to SQL Server, PostgreSQL or other relational databases depending on the environment.
