# iFood Marketing Analytics

## Sobre o Projeto

O projeto **iFood Marketing Analytics** foi desenvolvido com o objetivo de analisar dados de clientes, consumo e campanhas de marketing, transformando dados em insights estratégicos para apoio à tomada de decisão.

A análise contempla tratamento de dados, consultas SQL, análise exploratória em Python e construção de um dashboard executivo em Power BI.

---

## Objetivo

O objetivo principal é entender o perfil dos clientes, identificar padrões de consumo e avaliar o desempenho das campanhas de marketing.

Perguntas de negócio respondidas:

- Qual é o perfil dos clientes analisados?
- Quais segmentos geram maior consumo?
- Quais grupos respondem melhor às campanhas?
- Como o consumo varia por idade, renda e perfil familiar?
- Quais clientes possuem maior potencial para ações futuras?
