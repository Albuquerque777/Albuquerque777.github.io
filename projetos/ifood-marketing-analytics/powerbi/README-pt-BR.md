# Power BI - iFood Marketing Analytics

Esta pasta será utilizada para armazenar os arquivos e evidências do dashboard desenvolvido em Power BI para o projeto iFood Marketing Analytics.

## Objetivo do dashboard

Criar uma visão executiva para análise de clientes, consumo e campanhas de marketing.

## Páginas previstas

- Visão Executiva
- Perfil dos Clientes
- Análise de Consumo
- Campanhas de Marketing
- Segmentação de Clientes

## Indicadores principais

- Total de clientes
- Receita total analisada
- Gasto médio por cliente
- Renda média
- Taxa de resposta de campanha
- Total de compras
- Compras por canal
- Clientes de alto valor
- Clientes com maior potencial de campanha
