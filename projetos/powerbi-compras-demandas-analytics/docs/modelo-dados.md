# Modelo de Dados

## Visão geral

O projeto combina bases de Compras, Demandas e tabelas auxiliares para análise executiva.

## Bases principais

- Relatório de Compras: processos formais de contratação, aquisição ou cotação.
- Relatório de Demandas: oportunidades, projetos ou necessidades registradas para acompanhamento.
- Comparativo Compras x Demandas: análise de cobertura entre as duas bases.

## Cuidados de publicação

Antes de publicar no GitHub, revisar:

- Nomes reais de clientes
- IDs internos
- E-mails
- Valores confidenciais
- Dados comerciais sensíveis
- Conexões para SharePoint, OneDrive, SQL Server ou pastas corporativas
